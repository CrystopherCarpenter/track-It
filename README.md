
# Track It

A web app to manage your habits. Here you can record your habits, create new (good) habits, delete old ones and track your daily progression in executing them.

## How to run

1. Clone this repository
2. Install dependencies
```bash
npm i
```
3. Run the front-end with
```bash
npm start
```
4. You can optionally build the project running
```bash
npm run build
```
5. Finally on your browser access http://localhost:3000 (or the one indicated in the terminal after the execution of the npm start)
